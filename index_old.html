Computing

The course will use R for computing. R is freely available here. We recommend using R Studio which can be downloaded for free. We will use R Markdown as an authoring framework for creating reproducible data science documents.

Getting Started with R

  • If you have never programmed before then Hands-On Programming with R, by Garrett Grolemund is a great place to start.
  • R for Data Science, by Hadley Wickham and Garrett Grolemund is a wonderful resource.
  • A list of more resources is available on the course website here.

RStudio Sever or RStudio Desktop

  1. Students in the course will be able to do all computing in a server edition of RStudio. RStudio Server for STA130 students can be accessed by going to https://rstudio.chass.utoronto.ca/

  2. Another alternative is to install RStudio on your own computer. You will need to download R then RStudio.
